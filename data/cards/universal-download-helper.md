# Universal Download Helper

一个帮助快速下载网页中各类媒体资源的浏览器扩展/用户脚本。

## ✨ 主要功能
- 自动嗅探并列出当前页面所有可下载资源（图片、视频、音频等）
- 支持批量选择与下载
- 轻量、无广告、开源

## 🔧 安装与使用
1. 前往 [GitHub 仓库](https://github.com/WatchFleeting/universal-download-helper) 查看安装说明。
2. 根据你的浏览器安装对应版本（Chrome / Firefox / 油猴脚本）。
3. 打开任意包含媒体资源的网页，点击扩展图标即可开始抓取。

## ⚠️ 注意事项
- 请遵守相关网站的使用条款，仅下载你拥有合法权限的资源。
- 如遇问题，可在 GitHub Issues 中反馈。
